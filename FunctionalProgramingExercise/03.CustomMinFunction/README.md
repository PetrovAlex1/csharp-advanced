## Custom Min Function

Write a simple program that reads from the **console** a set of **integers** and **prints** back onto the **console** the **smallest**** number **from the collection. Use** Func\&lt;T, T\&gt;**.

### Examples

| **Input** | **Output** |
| --- | --- |
| 1 4 3 2 1 7 13 | 1 |
